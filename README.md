## How to Play

1. Use your mouse to orbit around the 3D scene
2. Click on the rotating cubes to score points
3. Each cube gives you random points (10-60)
4. New cubes appear after you click existing ones

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Technologies Used

- [Three.js](https://threejs.org/) - 3D graphics library
- [Vite](https://vitejs.dev/) - Build tool and development server
- Vanilla JavaScript
